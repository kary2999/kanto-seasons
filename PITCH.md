## 项目简介

**关东·四季 — KANTO SEASONS**：关东的四季出行。

「远方来信 · 日本系列」第 08 号。每天客户端按东京日期种子从内容池里选一条，每天本地 launchd 自动 commit + push 到 GitHub Pages，让 GitHub 提交记录每天多一格新绿色。

## 核心功能

- 15 处关东首都圈真实景点（含日光/箱根/那須等近郊）
- 四季流转粉绿配色 / 节气文艺
- 类 YORO·远方来信 美学
- 每条 Google Maps 跳转
- 每日定时器

## 技术栈

`HTML` · `Vanilla CSS` · `Vanilla JS` · `Python 3`（每日定时脚本）· `launchd`（macOS 定时器）· `GitHub Pages`

零依赖、零构建、零外部 API。一切都是静态文件 + 一份 JSON 内容池。
